Human-in-the-Loop for AI Agents

Enable your AI agents to interact with users through native system dialogs. Perfect for implementing human-in-the-loop workflows with MCP-compatible applications like Cursor and Claude Desktop.

Why Choose MCP User Feedback Server?

Seamlessly bridge the gap between AI automation and human input with our powerful MCP server.

Cross-Platform

Works seamlessly on macOS, Linux, and Windows with native system dialog integration.

Easy Integration

Simple one-line NPM installation and configuration with any MCP-compatible client.

Human-in-the-Loop

Enable AI agents to pause and request real-time user input during execution.

Quick Start

Get up and running in under 2 minutes

Installation 

npx mcp-user-feedback-server@latest

Configuration

Cursor IDE
{
  "mcpServers": {
    "user-feedback": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-user-feedback-server@latest"
      ]
    }
  }
}
Claude Desktop
{
  "mcpServers": {
    "user-feedback": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-user-feedback-server@latest"
      ]
    }
  }
}

Example 1: Simple User Input

Conversation

Tool Usage

  "question": "What is your favorite programming language?",
  "title": "User Preference"
}

Example 2: Form Input with Defaults

Conversation

User: Get my preferred email for notifications

AI: Let me ask for your preferred notification email.

Tool Usage
{
  "question": "Please enter your preferred email:",
  "title": "Email Settings",
  "defaultAnswer": "user@example.com"
}
