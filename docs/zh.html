为什么选择 MCP 用户反馈服务器？

使用我们强大的 MCP 服务器无缝连接 AI 自动化与人工输入。

跨平台支持

在 macOS、Linux 和 Windows 上无缝运行，集成原生系统对话框。

轻松集成

简单的一行 NPM 安装和配置，兼容任何 MCP 客户端。

人机协作

让 AI 智能体在执行过程中暂停并请求实时用户输入。

快速开始

不到 2 分钟即可运行

安装 

npx mcp-user-feedback-server@latest

配置

Cursor IDE
{
  "mcpServers": {
    "user-feedback": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-user-feedback-server@latest"
      ]
    }
  }
}
Claude Desktop
{
  "mcpServers": {
    "user-feedback": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-user-feedback-server@latest"
      ]
    }
  }
}

实际应用演示

人机协作工作流的真实案例

示例 1：简单用户输入

对话

用户：问我最喜欢的编程语言是什么

AI：我来问您最喜欢的编程语言。

工具使用
{
  "question": "您最喜欢的编程语言是什么？",
  "title": "用户偏好"
}

示例 2：带默认值的表单输入

对话

用户：获取我的通知邮箱偏好

AI：让我询问您的首选通知邮箱。

工具使用
{
  "question": "请输入您的首选通知邮箱：",
  "title": "邮箱设置",
  "defaultAnswer": "user@example.com"
}
